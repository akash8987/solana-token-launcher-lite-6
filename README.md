# Solana Token Launcher Lite 🚀

An expert-grade, beginner-friendly script to initialize, mint, and manage SPL Tokens on the Solana blockchain. This repository is structured for maximum readability and rapid deployment.

## Features
- **Token Creation**: Initialize a new SPL Token mint.
- **Metadata Support**: Simplified structure for On-chain metadata.
- **Minting**: Create a supply of tokens to a specific wallet.
- **Account Management**: Automatic creation of Associated Token Accounts (ATA).

## Quick Start
1. Install dependencies: `npm install`
2. Add your private key to `config.js`.
3. Run the launcher: `node launcher.js`

## Tech Stack
- Node.js
- @solana/web3.js
- @solana/spl-token
